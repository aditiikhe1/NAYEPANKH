# 🎓 NayePankh Foundation Student Management System

A Python-based Student Management System developed using Tkinter. This application helps manage student records efficiently through a user-friendly graphical interface.

---

## 📌 Project Description

The NayePankh Foundation Student Management System allows users to manage student information by performing various operations such as adding, updating, deleting, searching, and filtering student records. The system also generates reports in both TXT and PDF formats.

---

## ✨ Features

* Add Student Records
* Update Student Information
* Delete Student Records
* Search Students by Name
* Filter Students by Course
* Store Data in JSON File
* Automatically Load Saved Data
* Generate TXT Report
* Generate PDF Report
* Professional Dark-Themed GUI
* Student Dashboard with Total Student Count

## 📊 Functionalities

### Add Student

Add new student details including:

* Name
* Age
* Course
* Email

### Update Student

Modify existing student information.

### Delete Student

Remove selected student records.

### Search Student

Search students by name.

### Filter Students

Filter records based on course.

### Generate Reports

Generate TXT and PDF reports containing all student details.
